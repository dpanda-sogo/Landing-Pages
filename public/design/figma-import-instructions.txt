Use the free "html.to.design" Figma plugin to import the live HTML file:
1. Install "html.to.design" from the Figma Community (figma.com/community)
2. In Figma, open Plugins → html.to.design
3. Paste the URL of the Vercel preview page (or open the .html file locally in Chrome first)
4. The plugin converts every HTML element to native Figma layers, text styles, and fills
